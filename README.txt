========================================
DAIRY FARM MANAGEMENT SYSTEM
========================================

A comprehensive bilingual (English/Marathi) web application for managing dairy farm operations including workers, tasks, and cattle veterinary records.

========================================
QUICK START
========================================

1. Double-click setup.bat (Windows)
   OR
   Follow instructions in SETUP_INSTRUCTIONS.txt

2. Create admin account when prompted

3. Open browser: http://127.0.0.1:8000/

4. Login with admin credentials

5. Start managing your farm!

========================================
KEY FEATURES
========================================

- Bilingual Interface (English/Marathi)
- Admin & Worker Login System
- Worker Management
- Task Assignment with Deadlines
- Veterinary Section for Cows
- Medical Records Tracking
- Modern Responsive Design

========================================
TECHNOLOGY STACK
========================================

Backend: Django 4.2.7
Database: SQLite
Frontend: HTML, CSS, JavaScript
Language: Python 3.x

========================================
DOCUMENTATION
========================================

SETUP_INSTRUCTIONS.txt - Complete setup guide
PROJECT_SUMMARY.txt - Detailed project documentation
LANGUAGE_REFERENCE.txt - Translation reference

========================================
IMPORTANT
========================================

- Only Admin can create worker accounts
- No public registration available
- Keep admin credentials secure
- Regular database backups recommended
